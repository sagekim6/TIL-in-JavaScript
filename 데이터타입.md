# 데이터타입
* 숫자 타입  
  1.부동 소수점 형식을 다르기 때문에 모든 수를 실수로 처리한다.  
  ```javascript  
  console.log(1 === 1.0); // true  
  ```  
  2.Infinity(양의 무한대)  
  ```javascript  
  console.log(10/0); // infinity  
  ```  
  3.-Infinity(음의 무한대)  
  ```javascript  
  console.log(10/-0); // -infinity  
  ```
  4.NaN(not a number)  
  ```javascript  
  console.log(1*'문자열'); //NaN  
  ```  
  대소문자를 구분하기 때문에 꼭 NaN이라고 해야한다.  
  연산할 수 없는 문자열과 숫자를 연산하려고 하면 NaN(숫자가 아님)이라는 결과가 나온다.
